# Banque — Cryptographie

Codes et systèmes de chiffrement adaptés à 9-11 ans. Chaque entrée indique le niveau de difficulté et le matériel nécessaire.

---

## CODES DE SUBSTITUTION

### C01 — Code César
**Niveau**: Facile | **Matériel**: Papier + crayon (ou roue de déchiffrement)

Chaque lettre est décalée d'un nombre fixe de positions dans l'alphabet.
- Décalage standard: +3 (A→D, B→E, Z→C)
- Pour les 9 ans: fournir la roue de déchiffrement (un disque avec les deux alphabets alignés)
- Pour les 11 ans: donner uniquement le décalage en chiffre, construire soi-même

Exemple: `VRXV OH SLOLHU` (décalage 3) → `SOUS LE PILIER`

Variante: décalage différent pour chaque lettre (ex. A+1, B+2, C+3...) = code César progressif, niveau avancé.

---

### C02 — Code Atbash (miroir alphabétique)
**Niveau**: Facile-Moyen | **Matériel**: Tableau A=Z, B=Y...

Inversion complète de l'alphabet: A=Z, B=Y, C=X...
Intéressant car le déchiffrement utilise exactement le même processus que le chiffrement.

Exemple: `HLFH OZ GZYOV` → `SOUS LA TABLE`

Idéal comme "deuxième couche" d'un chiffrement en combinaison avec le César.

---

### C03 — Lecture en miroir
**Niveau**: Facile | **Matériel**: Miroir de poche (donné aux enfants au départ)

L'indice est écrit à l'envers, gauche/droite. Les enfants placent leur miroir contre le bord droit du message pour le lire.

Très efficace sur grand format (carte A4 posée à plat). Effet "waouh" garanti.

---

### C04 — Code Morse
**Niveau**: Moyen | **Matériel**: Table de référence Morse fournie (ou non, selon difficulté)

N'utiliser que les lettres du message à décoder. Transmettre le message:
- Sur papier (points et tirets)
- En frappant sur une table (court = point, long = tiret)
- En lumière de lampe de poche (très efficace le soir ou dans une pièce sombre)

Recommandation: limiter à 4-6 mots maximum. Fournir la table pour les 9 ans.

---

### C05 — Code numérique A=1, B=2
**Niveau**: Facile | **Matériel**: Papier

Chaque lettre remplacée par son rang dans l'alphabet.

Exemple: `19-15-21-19 12-5 16-15-18-10-5` → `SOUS LE PORJE`

Variantes pour augmenter la difficulté:
- A=2, B=4 (multiples de 2)
- A=3, B=6 (multiples de 3) — nécessite une division
- Nombres dans le désordre + indication de l'ordre à reconstituer

---

### C06 — Chiffre de Vigenère (simplifié)
**Niveau**: Difficile | **Matériel**: Table de Vigenère imprimée + mot-clé

Un mot-clé répété dicte le décalage de chaque lettre.
Exemple: mot-clé = LAPIN → L=décalage 12, A=1, P=16, I=9, N=14, puis recommencer.

À réserver comme épreuve finale ou bonus pour le 11 ans.

## CODES VISUELS ET SYMBOLIQUES

### C07 — Code Pig Pen (logettes)
**Niveau**: Moyen | **Matériel**: Grille de référence fournie (ou non)

Alphabet réparti dans deux grilles (carré + croix de Saint-André). Chaque lettre est représentée par la forme de sa case, avec ou sans point selon sa position.

Très visuellement mémorable. Les enfants adorent la "grille secrète". Fournir la grille de référence pour les 9 ans; la faire mémoriser aux 11 ans avant le début de la chasse.

---

### C08 — Code en formes géométriques
**Niveau**: Facile-Moyen | **Matériel**: Légende fournie

Créer une correspondance personnalisée: △=A, ○=B, □=C, ★=D...
Aucun standard — l'organisateur crée sa propre légende et la cache dans la maison (ex: sous un aimant sur le réfrigérateur, dans un livre spécifique).

Avantage: l'enfant doit d'abord *trouver* la légende avant de décoder le message.

---

### C09 — QR Code
**Niveau**: Facile (avec téléphone) | **Matériel**: QR code imprimé + téléphone

Générer un QR code sur [qr-code-generator.com](https://qr-code-generator.com) qui pointe vers:
- Un texte (le prochain indice)
- Une URL YouTube (courte vidéo du "lapin de Pâques")
- Une image (photo de la prochaine cachette en gros plan)

---

### C10 — Message à l'encre invisible
**Niveau**: Facile-Spectaculaire | **Matériel**: Jus de citron, coton-tige, sèche-cheveux

Écrire le message avec du jus de citron. Sécher. La feuille semble vierge.
Les enfants chauffent le papier avec le sèche-cheveux: le message apparaît progressivement en brun.

Équivalents possibles: lait, vinaigre blanc (variantes moins efficaces mais disponibles).
Effet dramatique garanti — à utiliser pour un indice clé ou la révélation finale.

## CODES STRUCTURELS

### C11 — Acrostiche
**Niveau**: Moyen | **Matériel**: Texte à lire

Un message innocent en apparence, dont les premières lettres de chaque ligne forment le vrai message.

Exemple:
> **S**elon la légende, le lapin arrive la nuit.
> **O**n dit qu'il cache ses trésors dans des lieux insolites.
> **U**ne chose est sûre: il faut chercher sous...
> **S**es traces commencent toujours par les fondations.

→ Premières lettres: S-O-U-S → SOUS

Variante: premières lettres de chaque mot au lieu de chaque ligne.

---

### C12 — Grille de lecture (carton perforé)
**Niveau**: Moyen | **Matériel**: Carton perforé + texte leurre

Préparer un carton avec des fenêtres découpées. Quand posé sur un texte "banal" (paragraphe d'histoire, recette de cuisine...), seules les lettres visibles à travers les fenêtres forment le vrai message.

Nécessite une préparation soignée. Effet très fort.

---

### C13 — Déchiffrement à deux clés (épreuve collaborative)
**Niveau**: Moyen | **Matériel**: Deux demi-légendes sur papier séparé

La légende de déchiffrement est coupée en deux:
- Enfant A reçoit la correspondance des lettres A à M
- Enfant B reçoit la correspondance des lettres N à Z

Ils doivent s'asseoir côte à côte et décoder ensemble. Ni l'un ni l'autre ne peut finir seul.
